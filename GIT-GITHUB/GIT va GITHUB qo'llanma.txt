https://github.com/signup?user_email=&source=form-home-signup

5. CMD qora oynasini ochamiz va u yerdan yuqoridagi papkani ichiga kiramiz.
6. git init
7. git config --global user.name "PalonchaPismadoncha"
8. git config --global user.email kimdirbirnima@gmail.com
9. 
git commit -m "biron nima deb yozing"
git branch -M main
git remote add origin https://github.com/PalonchaPismadoncha/DKREP.git
git push -u origin main
10. github.com saytiga o'tib saxifani yangilaymiz va fayllar qo'shilgan bo'ladi.
